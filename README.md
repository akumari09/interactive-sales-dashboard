# Interactive Sales Performance Dashboard (Excel)

## Project Overview
This project is an interactive sales dashboard created using Microsoft Excel.

## Features
- Pivot Tables
- Pivot Charts
- Slicers for dynamic filtering
- KPIs (Total Sales, Profit, Orders)
- Month-wise performance analysis

## Tools Used
- Microsoft Excel
- Pivot Tables
- GETPIVOTDATA Function

## Purpose
To analyze sales performance dynamically and help in business decision making.
